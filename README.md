## ✨ Características

- 📌 Registro y control de recursos del aula (HDMI, Cables Ethernet y Cables de audio).
- 🔍 Búsqueda y filtrado por Matrículas.
- 🗂️ Gestión de inventario.
- 🧾 Reportes exportables.
- 🔐 Login
